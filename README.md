The development of an Inventory Management System involves integrating various technologies to 
ensure a robust and efficient solution. The system is built using Flask, a lightweight WSGI web 
application framework in Python, to handle the back-end operations and server-side logic. For the
database layer, Oracle Database is utilized, providing a powerful and scalable solution for storing 
and managing inventory data. The front-end of the system is designed using HTML, CSS, JavaScript, and 
Bootstrap to create a responsive and user-friendly interface. Additionally, PHP can be employed for server-side
scripting to facilitate dynamic content and interactions. Combining these technologies results in a comprehensive 
Inventory Management System that streamlines inventory tracking and management processes.
